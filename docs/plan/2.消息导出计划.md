# 消息导出计划

导出数据库

一、私聊数据 c2c_msg_table

*   ✅ 发送者QQ
*   ✅ 发送者头像
*   ✅ 发送时间
*   ✅ 文字消息
*   ⬜ 图片消息
*   ⬜ 回复消息
*   ⬜ 卡片消息
*   ⬜ 分享类消息
*   ⬜ 撤回消息
*   ⬜ 红包一类
*   ⬜ 闪照
*   ⬜ 语音/视频通话
*   ⬜ 文件类消息



二、群聊数据 group_msg_table
*   ⬜ 群名称
*   ⬜ 群头像
*   ⬜ 发送者QQ
*   ⬜ 发送者头像
*   ⬜ 发送时间
*   ⬜ 图片消息
*   ⬜ 回复消息
*   ⬜ 卡片消息
*   ⬜ 分享类消息
*   ⬜ 撤回消息
*   ⬜ 红包一类
*   ⬜ 语音/视频通话
*   ⬜ 文件类消息

需要挖掘的api：

*   ⬜ 1群头像api
*   ⬜ 2群名称api
*   ✅ 用户头像api
*   ⬜ 用户名称api


以上信息或许会从本地进行获取
